Preguntas

¿Qué es ODBC y para qué es útil?

ODBC significa Open Database Connectivity el cual es básicamente una interfaz de programación de aplicaciones de estandar abierto el cual sirve para poder acceder a una base de datos.

¿Qué es SQL Injection?

Es una consulta sql, el cual sirve como una petición sobre alguna acción de una base de datos.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Se puede evitar por medio de la delimitación de valores en las consultas, de igual forma el siempre verificar los datos que introduce el usuario y por último el escapar con los caracteres especiales utilizados para consultas sql
