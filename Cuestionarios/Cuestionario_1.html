BD vs DBMS

Preguntas y Respuestas


¿Qué es entonces una base de datos - DB?

Una base de datos es una colección organizada de cierto tipo de información la cual esta estructurada, y guardada típicamente guardada electronicamente en un sistema de computadora.

¿En qué casos es conveniente usar bases de datos?

Los casos más convenientes para usar una base de datos son cuando el tipo de información tiene que ser gestionada, organizada y modificada, al realizar las bases de datos facilitaría su control de información.

¿Qué es un sistema de gestión de base de datos - DBMS?

Un sistema de gestión de bases de datos es básicamente un conjunto de programas que permiten almacenar, modificar y extraer todo tipo de información de una base de datos, básicamente es un intertaz entre la base de datos con el usuario.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Tipos de funciones de DBMS:

Abstracción de los datos -> Ahorran a los usuarios ciertos detalles de un almacenamiento físico de los datos.

Independencia -> Sirve para poder cambiar un esquema ya sea lógico o físico.

Consistencia -> En ciertos casos no se logra poder eliminar la redundancia, por lo que es necesario que alguien vigile los datos.

Integridad -> Sirve para proteger datos ante fallos de software o datos de entrada por usuarios.

Seguridad -> No todos tienen acceso a todo tipo de información hay una cierta restricción.

Control de la concurrencia -> En la ciestión de versiones deben de haber varias, pero solo con uno con los datos suficientes o necesarios.
