Describe Material design
Foundation es un tipo de framework que sirve para poder hacer la interfaz de usuario más user-friendly, de igual forma darle colores que sean adecuados para el tipo de servicio que se está ofreciendo. Foundation también proporciona compenentes en javascript, css, tipos de letra, botontes, barras de navegación etc. este framework es ocupado por compañías importantes tales como Disney o Adobe.
