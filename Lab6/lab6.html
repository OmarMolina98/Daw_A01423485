    Preguntas

    ¿Cuáles son las diferencias entre los posibles valores de la propiedad position?

    Las principales diferencias entre estos valores son:

    absolute: Coloca elementos por valores mediante valores de atributos left, top, botto y right.

    static: El valor inicial o el que va por defecto

    relative: Mediante este valor se puede posicionar un elemento al flujo de una página.

    ¿Cuáles son los valores estándar para la propiedad visibility?

    visible: Sirve para hacer visibles o invisibles elementos

    hidden: Es un atributo global booleano que indicica si un elemento está o no está.

    collapse: Se utiliza para fusionar los bordes

    ¿Qué es el zIndex y para qué sirve?

    Básicamente sirve para ordenar a cierto tipos de elementos haciendo que se superposicionen haciendo que el z-Index siempre este debajo de los demás o sea que un z-index cubra a uno menor en cuestión de elemento.
